# 🐳 EC2 Docker-Compose 컨테이너 및 로그 명령어

> **[필독] Docker Compose 명령어 사용 조건**
> `docker compose` 명령어는 반드시 **`docker-compose.yml` 파일이 있는 디렉터리(폴더)로 이동(`cd`)한 후**에 실행해야 합니다.

---

## 1. Docker Compose 기반 명령어 (추천)
프로젝트 단위(Nginx, Tomcat 등)로 묶여 있는 컨테이너만 깔끔하게 필터링해서 확인하고 제어할 때 사용합니다.

### 📌 우리 프로젝트 컨테이너 목록만 보기
현재 컴포즈 파일로 실행 중인 서비스들의 상태만 모아서 봅니다.
```bash

docker compose ps
```

### 📌 톰캣 로그 실시간 추적 (tail -f 방식)
최근 10줄만 띄운 상태에서 새로 찍히는 로그만 실시간으로 확인합니다.
```bash

docker logs -t -tail 10 tomcat
```

### 📌 서버 전체에서 docker-compose.yml 파일 경로 찾기
```bash

find / -name "docker-compose.yml" 2>/dev/null
```
