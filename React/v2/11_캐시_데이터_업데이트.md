# React Query: 캐시 데이터 업데이트 

### 설명
> setQueryData를 통해 기존 캐시 데이터를 수정할 때, 기존 데이터(Previous State)와 수정된 데이터(Update State)를 
> 합치는 과정이 필요합니다. 이때 자바스크립트의 Spread 연산자(...)를 사용합니다.

### 코드 예시 및 분석
```typescript
// [1] 기존 캐시 데이터 (prevPost)
const prevPost = {
   id: 1,
   title: "기존 제목",
   content: "기존 내용",
   views: 10
};

// [2] 서버에서 받은 수정된 데이터 (updatePost)
// 보통 서버는 수정된 필드만 보내주는 경우가 많음
const updatePost = {
   title: "수정된 제목"
};

// [3] 합치기 실행 (Spread 연산자 활용)
const result = { ...prevPost, ...updatePost };

console.log(result);
```

### 실행 결과 (Result)
```json
{ 
  "id": 1, 
  "title": "수정된 제목", 
  "content": "기존 내용", 
  "views": 10 
}
```
