## Supabase 주의사항‼️
### Supabase 프로젝트는 7일간 사용하지 않을 경우 중단됩니다.
Supabase 무료 버전은 특정 데이터베이스에 약 7일간 아무런 접근(어떠한 요청)도 발생하지 않으면, 자동으로 프로젝트를 중단시킵니다.
이럴 경우 데이터 페칭이 정상적으로 이루어지지 않아 실습 과정에 문제가 발생할 수 있습니다.
따라서 만약 실습 과정에서 갑자기 데이터 페칭이 실패한다면 이는 Supabase 프로젝트의 중단 문제일 수 있으므로 다음 순서에 
따라 Supabase 프로젝트의 상태를 확인하고 다시 가동해야 합니다.

### 1. Supabase 프로젝트 중단 여부 확인하기
Supabase 프로젝트의 중단 여부는 Supabase 대시보드에서 확인할 수 있습니다.
만약 프로젝트에 아래 그림과 같이 "Project is paused"라는 메시지가 표시되어 있다면
프로젝트가 지금 중단 상태라는 의미입니다.

![](https://github.com/dididiri1/TIL/blob/main/React/v2/images/05_07.png?raw=true)

2. 중단된 프로젝트 다시 가동하기
중단된 프로젝트를 클릭해 상세 페이지에 접속하면 아래 그림처럼 프로젝트를 다시 가동할 수 있는 〈Restore project〉 버튼이 있습니다. 
이 버튼을 클릭하면 즉시 프로젝트의 가동 작업이 시작됩니다.

![](https://github.com/dididiri1/TIL/blob/main/React/v2/images/05_08.png?raw=true)
복구 과정은 사용자의 시스템 사양에 따라 최대 10~20분 정도의 시간이 소요될 수도 있습니다.
완료되면 데이터베이스 연결 및 API 요청 등 프로젝트의 모든 기능이 정상적으로 다시 작동합니다.
