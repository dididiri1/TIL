# 섹션 2. 터미널 익숙해지기

## 💻 터미널(Terminal) 기본 명령어 및 가이드

터미널 환경별 단축키와 주요 명령어를 정리한 문서입니다.

---

### ⌨️ 터미널 줄바꿈 (Multi-line)
터미널에서 명령어를 전송하지 않고 다음 줄로 넘어가고 싶을 때 환경별로 아래 단축키를 시도해 보세요.

| 방법 | 설명 | 대상 환경 |
| :--- | :--- | :--- |
| **`Shift` + `Enter`** | 기본 지원 | macOS iTerm2, WezTerm, Ghostty, Kitty 등 |
| **`Option` + `Enter`** | 설정 필요 | macOS 기본 Terminal.app |
| **`Alt` + `Enter`** | 일부 환경 | Windows 환경 |
| **`Ctrl` + `Enter`** | 일부 환경 | Windows Terminal |
| **`\` + `Enter`** | **가장 확실한 방법** | **모든 환경 (역슬래시 후 엔터)** |

---

### 🛠️ 운영체제별 기본 명령어 비교

운영체제에 따라 명령어가 다를 수 있으니 확인 후 사용하세요.

| 기능 | Windows PowerShell | Windows CMD | macOS (Unix) |
| :--- | :--- | :--- | :--- |
| **현재 위치 확인** | `pwd` | `cd` | `pwd` |
| **폴더 이동** | `cd 폴더명` | `cd 폴더명` | `cd 폴더명` |
| **상위 폴더 이동** | `cd ..` | `cd ..` | `cd ..` |
| **홈으로 이동** | `cd ~` | `cd %USERPROFILE%` | `cd ~` |
| **파일 목록 보기** | `ls` 또는 `dir` | `dir` | `ls` |
| **숨김 파일 포함** | `ls -Force` | `dir /a` | `ls -a` |
| **폴더 생성** | `mkdir 폴더명` | `mkdir 폴더명` | `mkdir 폴더명` |
| **파일 복사** | `copy 원본 대상` | `copy 원본 대상` | `cp 원본 대상` |
| **파일 이동** | `move 원본 대상` | `move 원본 대상` | `mv 원본 대상` |
| **파일 삭제** | `del 파일명` | `del 파일명` | `rm 파일명` |
| **폴더 삭제** | `rmdir -r 폴더명` | `rmdir /S 폴더명` | `rm -r 폴더명` |
| **파일 내용 보기** | `cat 파일명` | `type 파일명` | `cat 파일명` |
| **화면 지우기** | `cls` | `cls` | `clear` |
| **도움말** | `Get-Help 명령어` | `명령어 /?` | `man 명령어` |
| **터미널 종료** | `exit` | `exit` | `exit` |

---

### 💡 핵심 팁
* **Windows CMD:** `cd` 명령어만 입력하면 현재 위치가 표시됩니다.
* **경로 구분:** Windows는 역슬래시(`\`), macOS/Linux는 슬래시(`/`)를 사용합니다.
* **⚠️ 삭제 주의:** `rm -r`과 `rmdir /S`는 폴더 전체를 삭제하므로 매우 신중하게 사용해야 합니다.

### 🚀 추천 터미널 도구

#### 1. PowerShell 7 (Windows 추천)
* **What:** 기능이 제한적인 기본 CMD를 대체하는 강력한 셸.
* **Why:** 클로드 코드(Claude Code) 등 최신 개발 도구를 실행하기에 가장 적합합니다.
* **Link:** [PowerShell 7 설치하기](https://learn.microsoft.com/ko-kr/powershell/scripting/install/installing-powershell-on-windows)

#### 2. iTerm2 (macOS 추천)
* **What:** macOS 기본 터미널보다 강력한 기능을 제공하는 대체 앱.
* **Why:** 탭 관리, 화면 분할, 테마 커스터마이징 등 개발 편의성이 뛰어납니다.
* **Link:** [iTerm2 공식 문서](https://iterm2.com/documentation.html)
