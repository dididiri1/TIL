## 🤖 클로드 코드(Claude Code) 가이드

클로드 코드는 터미널 환경에서 직접 실행되는 AI 에이전트입니다. 설치 전 가격 정책과 사양을 반드시 확인하세요.

### 1. 설치 및 초기 설정
클로드 코드를 사용하려면 터미널에서 아래 과정을 순서대로 진행해야 합니다.

* **Node.js 설치:** 클로드 코드는 Node.js 환경에서 실행됩니다. (v18 이상 권장)
* **설치 명령어:** 터미널에 아래 명령어를 입력하여 설치합니다.
    ```bash
    npm install -g @anthropic-ai/claude-code
    ```
* **초기 설정:** 설치 후 `claude`를 입력하여 실행하고, Anthropic 계정으로 로그인(OAuth)하여 API 키를 연동합니다.
