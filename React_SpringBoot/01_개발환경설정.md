# 섹션 1 개발환경 설정

## React 프로젝트에서 npm start와 npm run dev의 차이점은?
### ✅ 1. npm run start
- 주로 Create React App(CRA)으로 만든 프로젝트에서 사용합니다.
- 기본 개발 서버 포트는 3000이다.
- 프로젝트 생성 명령어 예시:
```
npx create-react-app my-app
```
### ✅ 2. npm run dev
- 주로 Vite 기반 프로젝트에서 사용됩니다.
- 기본 개발 서버 포트는 5173이다.
- 프로젝트 생성 명령어 예시:
```
npm create vite@latest
```
